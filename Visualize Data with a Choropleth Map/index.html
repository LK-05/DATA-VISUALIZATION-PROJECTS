Designed and coded by Julian NF for freeCodeCamp's Visualize Data with a Choropleth Map project. Project built using plain-vanilla JavaScript, D3.js, topoJSON, SCSS, and two XMLHttpRequests for JSON files. No tooltip or legend librairies where used. First time building a Choropleth Chart and happy to report I finished it in two working days (9hrs + 8hrs), including research, debugging, and styling, as well as an ultimately ill-fated but valiant attempt to employ foreignObject to render the tooltips within the SVG.
